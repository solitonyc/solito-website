My Photo Lightbox

The lightbox below uses JS script to create the hover effects, modal box operation, and navigation. This lightbox and the main navigation bars were the most difficult part components of the site for me. I ran into a lot of conflicts with my existing classes. However, was able to identify all the conflicts using Chrome's inspect.

I really enjoyed deconstructing how the code works and refining it to work with my own images and existing styles. I resolved all the conflicts by either create new classes or adjusting some of my existing ones, of which some were doing nothing. I enjoyed playing with this code; it was fun!
